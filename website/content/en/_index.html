What is Kubeflow on NKP?

The Kubeflow on NKP project is an open source distribution of Kubeflow, dedicated to making deployments of machine learning (ML) workflows on Nutanix Kubernetes Platform simple, portable and scalable. Kubeflow on NKP project provides seamless integration with Nutanix services.
